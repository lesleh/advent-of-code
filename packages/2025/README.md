# Advent of Code 2025

Solutions for Advent of Code 2025 challenges.

## Structure

Each day's solution is in its own directory (`01/`, `02/`, etc.) with:
- `index.ts` - Part 1 solution
- `part-2.ts` - Part 2 solution (when applicable)
- `input.txt` - Puzzle input

## Running Solutions

```bash
# Run a specific day's solution
npm run day01
npm run day02
npm run day03
npm run day04

# Run part 2 solutions
npm run day02:part2
npm run day03:part2
npm run day04:part2

# Type check all solutions
npm run check
```

## Development

This package uses:
- TypeScript with strict mode
- ES2022 modules
- tsx for running TypeScript files
